Вопросы и ответы

Часто задаваемые вопросы

Что такое PhotoViewer?▼
PhotoViewer — бесплатное настольное приложение для Windows для просмотра и управления фотографиями через локальную сеть. Состоит из WPF клиента, ASP.NET Core сервера и PostgreSQL базы данных.
Нужен ли интернет для работы?▼
Нет. PhotoViewer работает полностью в локальной сети без интернета. Интернет нужен только для установки приложения и .NET 8.
Сколько стоит PhotoViewer?▼
PhotoViewer полностью бесплатен. Исходный код открыт на GitHub. Вы можете использовать и изменять приложение свободно.
Какие системные требования?▼
Windows 10 или 11 (64-bit), .NET 8 Desktop Runtime. Для сервера дополнительно PostgreSQL 16+. Архитектура x64.
Сколько компьютеров может подключиться?▼
Неограниченное количество. Любой ПК или ноутбук в локальной сети может подключиться к серверу, введя его IP-адрес.
